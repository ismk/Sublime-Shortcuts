SUBLIME NAVIGATION
__________________

ACTION              											 KEYSTROKE
======															===========
Switch tabs 												   ⌘ (CMD) + ⌥ (ALT) + Arrow Keys(l/r)

Search Bar                							           ⌘ (CMD) + p

Select Quadrant 											   ⌃ (CRLT) + [1,2,3,4]

Move File to Quadrant									       ⌃ (CRLT) + ⇧ (Shift) + [1,2,3,4]
